## pnpm

[点击前往 pnpm 官网][pnpm]

### 简介

pnpm 是一个快速、可靠且节省磁盘空间的包管理工具。

它与传统的 npm 和 Yarn 不同，采用全局存储和符号链接等来处理依赖项，解决了重复安装和磁盘占用的问题，并避免了直接修改依赖项的风险。
